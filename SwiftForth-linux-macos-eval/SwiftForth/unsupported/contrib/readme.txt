This directory contains user contributions to SwiftForth and is included
in the SwiftForth distribution with the hope that you may find it
useful. FORTH, Inc. does not provide support or documentation for these
user-supplied contributions.

Please post any comments or requests for help on these source files to
the "SFTalk" mailing list.  This moderated list is available by
subscription to anyone interested in SwiftForth. To subscribe, send a
blank e-mail to sftalk-request@forth.com with the word "subscribe" as
the subject.  IMPORTANT: If you use Outlook, be sure to turn OFF "Remove
extra line breaks" in the Email Options settings. Otherwise, the mangled
reply cannot be processed by the list server.
