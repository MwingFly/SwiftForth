This is the original directory tree for the Cocoa-for-SwiftForth distribution.
It shows the file names with the revision digits I use at my end.
The FORTH, Inc. distribution ommits the revision digits to comply with their versioning system.
For the distro version I use the Dutch date format, yyyy-mm-dd, plus possible extra revision digit(s).
The date stands for the cocoa brew date as shown at startup.
